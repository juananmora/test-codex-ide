# Per-Run Plans

Each multi-agent workflow should create a new Markdown file in this folder.

Recommended file name pattern:

- `YYYY-MM-DD_HH-mm-ss_<workflow>.md`

Examples:

- `2026-03-20_11-40-00_review-main.md`
- `2026-03-20_11-52-10_bug-triage.md`

`docs/plan.md` should always point to the latest run and summarize its current status.
